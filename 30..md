#数据类型格式化说明符
```
数据类型       oc关键字     格式说明引导符

整型            int                  %d.%i

短整型          short int         %hd.%hi

长类型          long int           %ld.%li

无符号短整型     unsigned int        %u

无短整型        unsigned short      %hu

无符号长整型     unsigned long       %lu

浮点型          float                %f

双精度型        double               %f

长双精度型      long double          %lf

字符型           char                %c
```
